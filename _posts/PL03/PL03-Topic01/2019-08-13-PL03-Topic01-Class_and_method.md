## **Class advanced**
### ***__dict__***
```python
class DICT:
    def __init__(self, a, b):
        self.a = a
        self.b = b
    
CLA=DICT(1,2)
CLA.__dict__
```
<br><br><br>

### ***__str__***
```python
class STR():
    def __str__(self):
        return 'hello'
    
CLA=STR()
print(CLA)
```
