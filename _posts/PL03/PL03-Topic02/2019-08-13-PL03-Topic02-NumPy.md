## The numpy array object

```
>>> import numpy as np
>>> data = np.array([[1, 2],[3, 4],[5, 6]])
>>> type(data)
numpy.ndarray

>>> data
array([[1, 2],
       [3, 4],
       [5, 6]])
       
>>> data.ndim       
2

>>> data.shape
(3, 2)

>>> data.size
6

>>> data.dtype
dtype('int32')

>>> data.nbytes
24
```
