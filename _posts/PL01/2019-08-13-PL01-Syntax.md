## **C language basic syntax**

### ***Hello, World!, Basic***

```c
#include <stdio.h>

int main(){
    printf('Hello, World!\n');
    system('pause');
    return 0;
}
```
```
Hello, World!
```

#include <stdio.h>

int a;

int main(){

        system('pause');
        return 0;
}
